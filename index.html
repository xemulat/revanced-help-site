Sourcing the APK

Remember to check your architecture and DPI!

For example, Facebook Messenger has a lot of APKs available for archs (armeabi-v7a, x86, arm64-v8a) and DPI (360-480, 560-640, 120-160, 280-320, nodpi), still, universal | nodpi is your best bet. You have to download the "normal" APK, not the bundle version of the APK, bundles generally end with .apks, .xapk, .aab. They will generally be marked with the BUNDLE tag.

  1. ‚≠ê Search on the official website
  2. Search on google, the example query is: YouTube (version from here) apkmirror download

Fixes

The author of these fixes is taku_the_sixteenth on ReVanced's Discord server

  1. YouTube playback issue

    Recently YouTube has rolled out changes that affect the playback of videos. The Client spoof patch has been updated to work around this issue.

    • If you experience playback issues, patch YouTube using the latest version of ReVanced Patches. Make sure to include the Client spoof patch.
    • Make sure it is enabled under Misc > Spoof app signature.

  2. Rif solution for login problems:

    make sure manager version is 1.6.1

    reset your sources in manager settings (and restart manager)

    patch fresh apk of rif, include client spoof patch

    if installs fails, try uninstalling old rif first

    in case the above solution doesn't work:

    • make sure manager version is 1.6.1
    • in Manager settings > change sources "patches organization" field to Revanced (instead of revanced)
    • patch fresh APK of rif, include client spoof patch
    • if install fails, try uninstalling old rif first
